Data Encryption Standard (DES) is an outdated symmetric key method of data encryption. It was adopted in 1977 for government agencies to protect sensitive data and was officially retired in 2005. IBM researchers originally designed the standard in the early 1970s. DES stands for Data Encryption Standard. There are certain machines that can be used to crack the DES algorithm. The DES algorithm uses a key of 56-bit size. Using this key, the DES takes a block of 64-bit plain text as input and generates a block of 64-bit cipher text. The DES process has several steps involved in it, where each step is called a round. Depending upon the size of the key being used, the number of rounds varies.For example, a 128-bit key requires 10 rounds, a 192-bit key requires 12 rounds, and so on.
